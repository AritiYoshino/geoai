# GeoAI 项目命令说明

本命令用于快速提醒当前项目的关键入口、运行方式和文档位置。

## 项目定位

GeoAI ACE WebGIS 是一个基于 ACE 机制的多 Agent 地理空间分析系统，包含：

- WebGIS 地图问答系统。
- GIS 工具调用与受控空间代码执行。
- Critic / Evolution / Experience Library 闭环。
- 四组对比实验。
- 论文证据汇总接口。

## 常用入口

```bash
python main.py
```

默认访问：

```text
http://127.0.0.1:8000
```

页面：

- `/`：统一入口首页
- `/gis`：地理智能系统
- `/experiment`：对比实验系统
